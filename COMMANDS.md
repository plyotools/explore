# Quick Commands

## Save (Local Commit)
```bash
git save
```
Commits all changes to local main branch with message "Save changes"

## Release (Push to GitHub)
```bash
git release
```
Pushes the current branch to GitHub (origin main)

## Usage Examples

```bash
# Make some changes, then:
git save          # Commit locally
git release       # Push to GitHub (triggers deployment)
```
